Import Contacts
Load a CSV file for importing users in to the system. The CSV file needs to be formatted as a table and must include a header, or first line, that defines the fields in your table and contains each element as shown in the example below.
Gmail CSV Format
E-mail 1 - Value,Name
user1@knappsack.com,John Doe
user2@knappsack.com,Jane Doe
Outlook CSV Format
E-mail Address,First Name, Last Name
user1@knappsack.com,John,Doe
user2@knappsack.com,Jane,Doe
